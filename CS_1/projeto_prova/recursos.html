Recursos Úteis

Aqui compartilho uma lista de ferramentas, plataformas e leituras que considero valiosas para desenvolvimento, aprendizado e produtividade.

Desenvolvimento Web

  • MDN Web Docs (Mozilla Developer Network):

    Uma documentação completa e confiável para tecnologias web como HTML, CSS e JavaScript. Essencial para qualquer desenvolvedor.

    Visitar MDN

  • freeCodeCamp:

    Plataforma interativa para aprender a programar gratuitamente, com foco em desenvolvimento web e certificações.

    Visitar freeCodeCamp

  • Stack Overflow:

    A maior comunidade online para programadores tirarem dúvidas e compartilharem conhecimento.

    Visitar Stack Overflow

Produtividade e Organização

  • Notion:

    Um espaço de trabalho tudo-em-um para anotações, gerenciamento de projetos, wikis e bancos de dados. Extremamente flexível.

    Visitar Notion

  • Trello:

    Ferramenta de gerenciamento de projetos baseada em quadros Kanban, ótima para organizar tarefas visualmente.

    Visitar Trello

Leituras e Aprendizado Contínuo

  • Medium:

    Plataforma com uma vasta quantidade de artigos sobre diversos temas, incluindo tecnologia, startups e desenvolvimento pessoal.

    Visitar Medium

  • Udemy:

    Plataforma de cursos com preços acessíveis na área de programação, desenvolvimento e tecnologia.

    Visitar Udemy
